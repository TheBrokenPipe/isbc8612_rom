# Intel iSBC 86/12 Monitor ROM
This repo contains a toolchain for compiling the original Intel iSBC 86/12
monitor ROM, as well as the sources transcribed from the listing at the back
of the Intellec-iSBC 86/12 Interface and Execution Package User's Guide.

You will need a UNIX-like environment to build the ROM. Pre-built artifacts
are in the [`output`](./output) directory for people who do not have a
UNIX-like environment.

To build, run:

```sh
bash build.sh
```

The output will be saved to [`output/bin/SBCMON.BIN`](./output/bin), with
objects and listings in [`output/obj`](./output/obj).

Note that this toolchain uses PL/M-86 Compiler version 1.2, not the version
1.0 Intel used, because version 1.0 is not available. This will not produce
a ROM image identical to the original (which is not available anyway),
because version 1.2 has better optimisations and generates smaller code.

Please also note that the transcription may not be 100% accurate. I've only
verified that the characters are in the correct locations using a mask
generated by segmenting the listing scans, and that the monitor's basic
functionalities are present. If you notice a transcription error, please open
an issue.
